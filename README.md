# נור א-שמס — דגם תלת־ממדי מהמפות

אפליקציית ווב שבונה דגם תלת־ממדי של נור א-שמס (טול כרם) מתוך מספר מפות של אותו מקום:
מפה טופוגרפית, מפת כבישים, תצלום אוויר כללי ו-6 תצלומי אוויר חלקיים ברזולוציה גבוהה.

**צפייה:** https://chepti.com/maps/

## איך זה עובד — הצינור (pipeline/)

1. **reg.py** — רישום אוטומטי של תצלומי האוויר החלקיים על התצלום הכללי
   (התאמת תבניות רב-סקאלית על שדה גרדיאנטים).
2. **reg3b.py / mosaic.py** — התאמת מאפיינים
   (SIFT + RANSAC)
   בין התצלומים החלקיים, פתרון ריבועים-פחותים למיקומים, ובניית פסיפס ברזולוציה מלאה
   (0.38 מ'/פיקסל)
   עם מיזוג feather בתפרים.
3. **dem.py / dem2.py** — בניית מודל גבהים מהמפה הטופוגרפית:
   - מדידת רשת הק"מ (הריבועים השחורים) → 2.98 מ'/פיקסל.
   - חילוץ מסכת קווי הגובה החומים לפי צבע.
   - משטח ראשוני מנקודות בקרה שנקראו מהמפה (תוויות 120/180/200, נ.ג. 167, פרופיל הוואדי).
   - הצמדה: כל רכיב מחובר של קו גובה מוצמד לכפולת 10 מ' הקרובה, ואינטרפולציית
     RBF (thin-plate)
     מחדש מאלפי הנקודות המוצמדות.
4. **buildings.py** — זיהוי מבנים מהפסיפס (בהירות/רוויה/צמחייה + מורפולוגיה),
   זיהוי אוטומטי של כיוון הצל (דרום-מערב → צילום בוקר), והערכת גובה יחסי לפי אורך הצל.
5. **compose.py** — יישור כל השכבות למסגרת משותפת לפי סמני העיגולים הכחולים
   (הווקטור בין שני הסמנים זהה בין המפה הטופוגרפית לתצלום — סקאלה 1.013, ללא סיבוב),
   הרכבת טקסטורות, וייצוא נתונים לאפליקציה.

## האפליקציה (webapp/)

Three.js
ללא תלות חיצונית (הכל מקומי). רשת גבהים 262×196 + חצאית דיורמה, טקסטורת תצ"א 4096px,
שכבות טופו/כבישים/קווי-גובה במיקסר שיידר, בתים כ-InstancedMesh
עם גובה מהצללים. עובד במגע (סיבוב/צביטה) — מותאם לטלפון.

## הרצה מקומית

```
python serve.py   # שרת על פורט 8123 + endpoint לצילומי דיבוג
```
