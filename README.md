# דגם תלת־ממדי מהמפות

תוכנה שמקבלת תיקיית מפות של מקום (מפה טופוגרפית, מפת כבישים, תצלומי אוויר) ובונה דגם
תלת־ממדי אינטראקטיבי — סיבוב, זום והטיה גם בטלפון, בלי להתקין כלום.

**צפייה:** https://chepti.com/maps/

## הוספת יעד חדש (סט תמונות אחר)

1. שמים את כל התמונות בתיקייה אחת: מפה טופוגרפית, תצלום אוויר כללי, תצלומי אוויר
   חלקיים (כמה שיותר — רזולוציה טובה יותר), ואופציונלית מפת כבישים.
   התוכנה מזהה לבד מה זה מה (מפות מצוירות הן "שטוחות", תצלומים מלאי מרקם).
2. יוצרים קובץ קונפיג עם נקודות גובה מהמפה הטופוגרפית (ראו
   `pipeline/nur-shams.config.json`
   כדוגמה): כל שורה היא
   `[x_פיקסל, y_פיקסל, גובה_במטרים]`
   לפי תוויות קווי הגובה שעל המפה. בלי זה — השטח יצא שטוח.
3. אם יש עיגולים כחולים מסומנים על המפה הטופוגרפית ועל התצ"א (שתי נקודות זהות בשתיהן) —
   היישור ביניהן אוטומטי. אחרת מוסיפים לקונפיג
   `"topo_to_overview": [scale, ox, oy]`.
4. מריצים:

```
python pipeline/run_all.py --input <תיקיית_התמונות> --site <מזהה-באנגלית> --name "<שם לתצוגה>" --config pipeline/<מזהה>.config.json
```

5. מעלים לשרת את
   `webapp/data/<מזהה>/`
   ואת
   `webapp/data/sites.json`
   — האתר יציג בורר אתרים אוטומטית.

## מה הצינור עושה (pipeline/run_all.py)

1. **סיווג** — מזהה אוטומטית מפה טופוגרפית / מפת כבישים / תצלומי אוויר לפי שטיחות ומרקם.
2. **רישום** — ממקם כל תצלום חלקי על התצלום הכללי (התאמת תבניות רב-סקאלית), ואז מדייק
   בהתאמת מאפיינים
   (SIFT + RANSAC)
   בין התצלומים ופתרון ריבועים-פחותים. בונה פסיפס ברזולוציה מלאה (~0.37 מ'/פיקסל).
3. **מודל גבהים** — מודד את רשת הק"מ במפה (ריבוע שחור = 1 ק"מ), מחלץ את קווי הגובה
   החומים (צבע + מסנני צורה: פס דק וארוך), מצמיד כל קו לכפולת 10 מ' לפי משטח ראשוני
   מנקודות העיגון, ובונה שטח רציף באינטרפולציית
   RBF.
4. **יישור שכבות** — לפי סמני העיגולים הכחולים המשותפים.
5. **מבנים** — מזהה גגות (בהירות/רוויה/מרקם), חותך כל מבנה למלבן מסובב לפי צורתו,
   מזהה את כיוון הצל אוטומטית ומחשב גובה אישי לכל מבנה לפי אורך הצל שלו.
6. **צמחייה** — אינדקס ירוק עודף
   (ExG)
   מזהה עצים ושיחים; עץ (עם צל / כתם גדול) מקבל גובה 3.5–9 מ', שיח 1–2 מ'.

## האפליקציה (webapp/)

Three.js
מקומי בלבד (בלי CDN). שטח 262×196 קודקודים עם דפנות דיורמה כהות, טקסטורת תצ"א
4096px, מיקסר שכבות בשיידר (תצ"א/טופו/כבישים/קווי גובה), מבנים וצמחייה כ-InstancedMesh.

- **הגזמת גובה** (×1–×5) פועלת על הטופוגרפיה בלבד — מבנים ועצים שומרים על גובהם האמיתי.
- בטלפון: הפאנל מוסתר, כפתור צף 🎛 פותח וסוגר אותו.

## הרצה מקומית

```
python serve.py   # http://localhost:8123 + POST /save לצילומי דיבוג
```
